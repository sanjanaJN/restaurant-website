# 🍝 La Bella Cucina – Restaurant Website

A modern multi-page restaurant website built using HTML, CSS, and JavaScript.

## 🚀 Features
- Responsive design
- Dynamic menu rendering
- Add to cart functionality
- LocalStorage cart persistence
- Remove and clear cart options
- Multi-page navigation

## 📍 Location
UB City, Bangalore, India

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## 📌 How to Run
1. Clone the repository
2. Open index.html in browser
3. Use Live Server for better experience
